# Mobirio ベンダーダッシュボード UI/UX 改修計画

> 作成日: 2026-02-19
> ステータス: 計画承認済み・段階実装中

---

## 背景

現行のベンダー管理画面が参考にした競合企業（moto-auc RENTAL BIKE Backyard）のデザインに類似しすぎており、差別化が必要。ユーザーの理想は「直感的に使えるUI/UX」。

### 参考画像との類似度

| 要素 | 類似度 |
|------|--------|
| ナビゲーション項目・順序（10カテゴリ/25項目） | **完全一致** |
| サイドバー構造（左固定260px、アコーディオン式） | 極めて高い |
| ダッシュボードTOP配置（統計→出発予定→3列カード） | 極めて高い |
| 一覧ページ構成（パンくず→検索→テーブル→ページネーション） | **完全一致** |
| ボタン配置（「+ 新規」「CSV出力」のタイトル右配置） | **完全一致** |
| 車両カレンダー（ガントチャート方式） | 極めて高い |
| 車両フォーム（12セクション縦積み） | 極めて高い |
| ライダーズギア一覧（カラム構成・複製ボタン含む） | **完全一致** |

---

## 差別化の3本柱

| 柱 | 概要 | 競合との違い |
|---|------|------------|
| **操作体験の革新** | コマンドパレット、インライン編集、D&D、ショートカット | 競合=古典的CRUD画面 |
| **情報の可視化** | ステータスボード、タイムライン、ヒートマップ、カンバン | 競合=テーブル一辺倒 |
| **実運用フィット** | クイック予約、メンテアラート、チェックリスト、CRM | 競合=データ管理止まり |

## 配色方針

### 採用方向: ウォームニュートラル

| 要素 | 現行 | 変更後 |
|------|------|--------|
| ベース背景 | `#F9FAFB`（gray-50） | `#FAFAF8`（ウォームホワイト） |
| サーフェス背景 | `#FFFFFF` | `#F5F3EF`（ウォームグレー） |
| テキスト | `#374151`（gray-700） | `#2D2D2D`（ソフトブラック） |
| アクセント | `#2D7D6F`（ティールグリーン） | `#D4763C`（テラコッタオレンジ） |
| アクセントダーク | `#1A5A4F` | `#B85A2B` |
| アクセントライト | `#3D9D8F` | `#E8945A` |
| サブカラー | なし | `#6B8E6B`（セージグリーン） |
| テーブルヘッダー | `bg-gray-50` | `bg-accent/5`（アクセント極薄ティント） |

### 理由
- バイク = アウトドア = 暖かみ、というブランドイメージと一致
- Backyardの「冷たい青/緑」と明確に差別化
- 業界特有のアイデンティティを表現

## ナビゲーション方針

### 採用方式: コンパクトアイコンレール + スライドパネル + コマンドパレット

```
通常時:
[アイコンレール 60px] [メインコンテンツ（広い）]

ホバー/クリック時:
[アイコンレール 60px] [パネル 240px] [メインコンテンツ]

Cmd+K:
[コマンドパレットモーダル: 全ページ検索 + アクション実行]
```

### サイドバーカテゴリ（アイコン）
- 🏠 TOP
- 📋 レンタル管理
- 🏍 車両管理
- 🪖 ギア管理
- 🏪 店舗管理
- 📊 分析
- 📦 データ出力
- 📢 お知らせ
- ⚙️ 設定

### よく使うメニュー
- ピン留め機能でカスタマイズ可能
- デフォルト: TOP、予約一覧、車両一覧、カレンダー
